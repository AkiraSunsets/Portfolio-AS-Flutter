Um aplicativo de portfólio pessoal desenvolvido em **Flutter** para destacar habilidades em desenvolvimento de software e UI/UX design. Este projeto apresenta de forma interativa e fluida a minha trajetória, experiências e projetos concluídos.

---

## 🎯 Objetivo do Projeto

O objetivo deste aplicativo é centralizar e exibir meus trabalhos técnicos e acadêmicos em uma interface moderna e responsiva. Ele serve como um currículo dinâmico, permitindo que recrutadores e outros desenvolvedores explorem minhas habilidades e vejam detalhes dos projetos em que atuei.

## 🚀 Como Executar

### Pré-requisitos

<div align="center">

![Flutter SDK](https://img.shields.io/badge/Flutter%20SDK-Required-blue?style=flat-square&logo=flutter&logoColor=white)
![Emulator](https://img.shields.io/badge/Emulator-Android%20%7C%20Web%20%7C%20Windows-brightgreen?style=flat-square)

</div>

- [Flutter SDK](https://docs.flutter.dev/get-started/install) instalado na sua máquina
- Um emulador configurado (Android) ou navegador web
- Dart SDK instalado

### Instalação

1. **Clone o repositório**
```bash
git clone https://github.com/seu-usuario/Portfolio-AS-Flutter.git
cd Portfolio-AS-Flutter
```

2. **Instale as dependências**
```bash
flutter pub get
```

3. **Execute o aplicativo**
```bash
flutter run
```
